# Lair Scanner

> File scanner for the [Lair](https://github.com/AlphaHydrae/lair) media center.

[![NPM version](https://badge.fury.io/js/lair-scanner.svg)](http://badge.fury.io/js/lair-scanner)



## Installation

```bash
npm install -g lair-scanner
```

## Usage

```bash
lair
```
